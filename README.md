# CTES Rumkin

This crate is used to host all logic for porting Rumkin.com cipher logic, and generating bindings for various programming languages.

## Credit

A majority of the cipher-related source code is a port of [fidan/rumkin](https://github.com/fidian/rumkin/tree/master/site/tools/cipher), for educational use. The port intends to integrate Rumkin into the CTES (Ciphertext Ecosystem).
